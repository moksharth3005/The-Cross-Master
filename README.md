The game is a simple 2D shooter in which enemies will spawn at the right corner of the screen the enemies cannot kill you if you touch them but if they succeed in reaching the left corner of the screen then it is a game over

controls:-

left/right/up/down - to move
space bar- shoot a bullet
to make the game harder you only have one bullet but you can call the bullet back by pressing the spacebar again

The game is now released, v0.1.1 is the stable release version of the game

V0.0.1, V0.0.2, V0.0.3 were all early access

version 0.1.1 is the stable release of the game with probably one more update coming in the near future adding gamepad support
